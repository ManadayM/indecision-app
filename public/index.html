Install babel-cli@6.24.1 as Global dependency through yarn.

Init yarn

yarn add babel-preset-react@6.24.1 babel-preset-env@1.5.2

create new folder "src" in the root of the project that will keep source code while public/scripts will hold autogenerated transpiled code from Babel.

babel src/app.js --out-file=public/scripts/app.js --presets=react,env --watch
